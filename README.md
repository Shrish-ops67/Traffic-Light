# 🚦 Traffic Light using HTML & CSS

A simple Traffic Light UI built using only HTML and CSS. This project is beginner-friendly and demonstrates the basics of HTML structure and CSS styling.

## 📌 Features
- Pure HTML and CSS
- Responsive and clean layout
- Three colored traffic lights (Red, Yellow, Green)
- Rounded container design
- Beginner-friendly code

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📂 Project Structure
```
Traffic-Light/
│── index.html
│── style.css
└── README.md
```

## 🚀 How to Run
1. Clone this repository:
   ```bash
   https://github.com/Shrish-ops67/Traffic-Light
   ```

2. Open the project folder.

3. Double-click `index.html` or open it in your preferred web browser.

## 📸 Preview
The webpage displays a traffic light with:
- 🟢 Green Light
- 🟡 Yellow Light
- 🔴 Red Light

## 📚 Learning Objectives
This project helps beginners understand:
- Basic HTML structure
- CSS selectors
- Styling with colors
- Border and border-radius
- Flexbox for layout
- Centering elements
